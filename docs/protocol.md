# Testing Protocol

Prereqs: `uv sync`, test ND2, output dir.

**Processing**: Launch → Browse ND2 → configure → Start Workflow → Cancel test → Complete → Merge (samples.yaml + processing_results.yaml)

**Visualization**: Load Folder → select channels → Start → navigate traces, toggle good/bad → Save Inspected CSV

**Analysis**: Load CSV → pick model → Start Fitting → quality panel, parameter analysis, Save All Plots

**Cross-tab**: Full flow; tab switching; error handling for invalid files
